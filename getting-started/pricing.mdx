For those who prefer a hassle-free, managed solution, we offer **Chonkie Cloud**. This is a hosted API service that lets you leverage Chonkie's chunking power without managing the infrastructure.

-   **Usage-based Pricing:** Chonkie Cloud operates on a pay-as-you-go model.
-   **API Key Required:** Access requires an API key, which you can obtain from the [Chonkie Cloud dashboard](https://cloud.chonkie.ai/keys) after signing up.
-   **Simple Integration:** Easily integrate chunking into your applications via HTTP requests using our [Python client](/chonkie-cloud/overview#basic-usage) or standard tools like `cURL`.

For organizations requiring maximum control, data privacy, and dedicated support, we offer **Chonkie On-Prem**. This solution deploys the full power of Chonkie within your own infrastructure, managed with our expertise.

-   **Runs on Your Infrastructure:** Deploy Chonkie entirely within your network or private cloud, ensuring data never leaves your control.
-   **Full Data Sovereignty & Security:** Meets stringent compliance and security requirements by keeping all processing internal.
-   **Dedicated Support:** Benefit from premium support, including a dedicated Slack channel for direct access to our team.
-   **Managed Updates & Maintenance:** We handle the updates and maintenance, ensuring you always have the latest features and security patches without the operational overhead.
-   **Scalability:** Scale the deployment according to your needs within your own environment.
-   **Predictable Pricing:** Typically involves custom contracts tailored to your usage and support needs.
